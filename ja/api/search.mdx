## クエリパラメータ

<ParamField query="q" type="string" required>
  検索クエリ
</ParamField>

<ParamField query="type" type="string">
  検索対象タイプ: `products`、`users`、`orders`、`all`（デフォルト: `all`）
</ParamField>

<ParamField query="page" type="integer" default="1">
  ページ番号
</ParamField>

<ParamField query="per_page" type="integer" default="20">
  1ページあたりの結果数（最大: 100）
</ParamField>

<ParamField query="filters" type="object">
  追加フィルタ条件
</ParamField>

<ParamField query="sort" type="string" default="relevance">
  ソート順: `relevance`、`date`、`price`、`name`
</ParamField>

```python Python
import requests

url = "https://api.example.com/api/v1/search"
headers = {"Authorization": "Bearer YOUR_API_KEY"}

# 商品を検索
params = {
    "q": "ワイヤレス ヘッドホン",
    "type": "products",
    "per_page": 30,
    "sort": "price"
}

response = requests.get(url, headers=headers, params=params)
results = response.json()

print(f"検索結果: {results['total_count']}件")
for item in results['results']:
    if item['type'] == 'product':
        print(f"- {item['name']}: ¥{item['price']:,}")
```

```javascript JavaScript
// 複数タイプを検索
const params = new URLSearchParams({
  q: 'ワイヤレス',
  type: 'all',
  per_page: '50'
});

const response = await fetch(
  `https://api.example.com/api/v1/search?${params}`,
  {
    headers: {
      'Authorization': 'Bearer YOUR_API_KEY'
    }
  }
);

const results = await response.json();

// 結果をタイプ別に分類
const byType = results.results.reduce((acc, item) => {
  acc[item.type] = (acc[item.type] || 0) + 1;
  return acc;
}, {});

console.log('検索結果:', byType);
```

```go Go
package main

import (
    "encoding/json"
    "fmt"
    "net/http"
    "net/url"
)

func search(query string) {
    baseURL := "https://api.example.com/api/v1/search"

    params := url.Values{}
    params.Add("q", query)
    params.Add("type", "products")
    params.Add("per_page", "20")

    req, _ := http.NewRequest("GET", baseURL+"?"+params.Encode(), nil)
    req.Header.Set("Authorization", "Bearer YOUR_API_KEY")

    client := &http.Client{}
    resp, _ := client.Do(req)
    defer resp.Body.Close()

    var result map[string]interface{}
    json.NewDecoder(resp.Body).Decode(&result)

    fmt.Printf("検索結果: %.0f件\n", result["total_count"])
}
```

## 検索演算子

<AccordionGroup>
  <Accordion title="フレーズ検索">
    `"ワイヤレス ヘッドホン"` - 完全一致フレーズを検索
  </Accordion>

  <Accordion title="AND検索">
    `ワイヤレス AND ノイズキャンセリング` - 両方の単語を含む
  </Accordion>

  <Accordion title="OR検索">
    `ワイヤレス OR 有線` - いずれかの単語を含む
  </Accordion>

  <Accordion title="NOT検索">
    `ヘッドホン NOT 有線` - 特定の単語を除外
  </Accordion>

  <Accordion title="ワイルドカード">
    `ワイヤ*` - 前方一致検索
  </Accordion>
</AccordionGroup>

<Tip>
検索結果はリアルタイムで更新されます。商品が追加または更新されると、即座に検索可能になります。
</Tip>
