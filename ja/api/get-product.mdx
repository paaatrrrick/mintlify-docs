---
title: "商品詳細の取得"
api: "GET /api/v1/products/{id}"
description: "商品の詳細情報を取得します"
---

## パスパラメータ

<ParamField path="id" type="string" required>
  取得する商品の一意の識別子
</ParamField>

## クエリパラメータ

<ParamField query="include" type="string">
  追加データを含める: `reviews`、`related`、`variants`（カンマ区切り）
</ParamField>

<RequestExample>
```bash cURL
curl -X GET "https://api.example.com/api/v1/products/prod_123?include=reviews,related" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```python Python
import requests

url = "https://api.example.com/api/v1/products/prod_123"
headers = {"Authorization": "Bearer YOUR_API_KEY"}
params = {"include": "reviews,related,variants"}

response = requests.get(url, headers=headers, params=params)
product = response.json()

print(f"商品名: {product['name']}")
print(f"価格: ¥{product['price']}")
print(f"在庫: {product['stock_quantity']}個")
print(f"レビュー数: {len(product.get('reviews', []))}件")
```

```javascript JavaScript
const response = await fetch(
  'https://api.example.com/api/v1/products/prod_123?include=reviews,related',
  {
    headers: {
      'Authorization': 'Bearer YOUR_API_KEY'
    }
  }
);

const product = await response.json();
console.log(`${product.name} - ¥${product.price.toLocaleString('ja-JP')}`);
```

```swift Swift
import Foundation

let url = URL(string: "https://api.example.com/api/v1/products/prod_123?include=reviews")!
var request = URLRequest(url: url)
request.addValue("Bearer YOUR_API_KEY", forHTTPHeaderField: "Authorization")

URLSession.shared.dataTask(with: request) { data, response, error in
    guard let data = data else { return }

    let product = try? JSONDecoder().decode(Product.self, from: data)
    print("商品: \(product?.name ?? "")")
}.resume()
```
</RequestExample>
