## ボディパラメータ

<ParamField body="order_id" type="string" required>
  支払いを処理する注文ID
</ParamField>

<ParamField body="payment_method" type="string" required>
  支払い方法: `credit_card`、`debit_card`、`bank_transfer`、`convenience_store`、`mobile_payment`
</ParamField>

<ParamField body="payment_details" type="object" required>
  支払い詳細
  <Expandable title="クレジットカードの場合">
    <ParamField body="card_number" type="string" required>
      カード番号
    </ParamField>
    <ParamField body="card_holder" type="string" required>
      カード名義人
    </ParamField>
    <ParamField body="expiry_month" type="string" required>
      有効期限（月）
    </ParamField>
    <ParamField body="expiry_year" type="string" required>
      有効期限（年）
    </ParamField>
    <ParamField body="cvv" type="string" required>
      セキュリティコード
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="save_payment_method" type="boolean" default="false">
  今後の使用のために支払い方法を保存
</ParamField>

<ParamField body="installments" type="integer">
  分割払い回数（オプション: 3、6、12、24）
</ParamField>

<RequestExample>

```python Python
import requests

url = "https://api.example.com/api/v1/payments"
headers = {
    "Authorization": "Bearer YOUR_API_KEY",
    "Content-Type": "application/json"
}
data = {
    "order_id": "order_123",
    "payment_method": "credit_card",
    "payment_details": {
        "card_number": "4111111111111111",
        "card_holder": "YAMADA TARO",
        "expiry_month": "12",
        "expiry_year": "2025",
        "cvv": "123"
    },
    "save_payment_method": True,
    "installments": 3
}

response = requests.post(url, json=data, headers=headers)
payment = response.json()

if payment['status'] == 'succeeded':
    print(f"支払いが成功しました: ¥{payment['amount']:,}")
    print(f"取引ID: {payment['transaction_id']}")
else:
    print(f"支払いが失敗しました: {payment['error_message']}")
```

## 支払い方法の種類

<AccordionGroup>
  <Accordion title="クレジットカード">
    Visa、Mastercard、JCB、American Express、Diners Clubに対応
  </Accordion>

  <Accordion title="コンビニ決済">
    セブンイレブン、ファミリーマート、ローソンで支払い可能
  </Accordion>

  <Accordion title="銀行振込">
    指定の銀行口座への振込。通常1〜3営業日で確認
  </Accordion>

  <Accordion title="モバイル決済">
    PayPay、LINE Pay、楽天ペイに対応
  </Accordion>
</AccordionGroup>

<Warning>
クレジットカード情報は暗号化されて送信されます。システムにはカード情報は保存されません。
</Warning>

<Tip>
分割払いは手数料無料でご利用いただけます（3回、6回、12回払い）。
</Tip>
