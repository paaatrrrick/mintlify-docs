## クエリパラメータ

<ParamField query="page" type="integer" default="1">
  ページ番号
</ParamField>

<ParamField query="per_page" type="integer" default="20">
  1ページあたりのアイテム数（最大: 100）
</ParamField>

<ParamField query="status" type="string">
  ステータスでフィルタ: `active`、`inactive`、`suspended`
</ParamField>

<ParamField query="role" type="string">
  ロールでフィルタ: `admin`、`user`、`moderator`
</ParamField>

<ParamField query="search" type="string">
  名前またはメールアドレスで検索
</ParamField>

<ParamField query="created_after" type="string">
  この日付以降に作成されたユーザーをフィルタ（ISO 8601形式）
</ParamField>

<ParamField query="sort_by" type="string" default="created_at">
  ソートフィールド: `name`、`email`、`created_at`
</ParamField>

<ParamField query="order" type="string" default="desc">
  ソート順: `asc`または`desc`
</ParamField>

<RequestExample>
```bash cURL
curl -X GET "https://api.example.com/api/v1/users?status=active&role=user&page=1&per_page=20" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```python Python
import requests

url = "https://api.example.com/api/v1/users"
headers = {"Authorization": "Bearer YOUR_API_KEY"}
params = {
    "status": "active",
    "role": "user",
    "page": 1,
    "per_page": 20,
    "sort_by": "created_at",
    "order": "desc"
}

response = requests.get(url, headers=headers, params=params)
print(response.json())
```


```java Java
import java.net.http.*;
import java.net.URI;

HttpClient client = HttpClient.newHttpClient();

String url = "https://api.example.com/api/v1/users" +
             "?status=active&role=user&page=1&per_page=20";

HttpRequest request = HttpRequest.newBuilder()
    .uri(URI.create(url))
    .header("Authorization", "Bearer YOUR_API_KEY")
    .GET()
    .build();

HttpResponse<String> response = client.send(
    request,
    HttpResponse.BodyHandlers.ofString()
);

System.out.println(response.body());
```

## 使用例

<CodeGroup>
```bash アクティブなユーザーを取得
curl -X GET "https://api.example.com/api/v1/users?status=active" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```bash 管理者を検索
curl -X GET "https://api.example.com/api/v1/users?role=admin" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```bash 名前で検索
curl -X GET "https://api.example.com/api/v1/users?search=山田" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```bash 最近作成されたユーザー
curl -X GET "https://api.example.com/api/v1/users?created_after=2024-01-01T00:00:00Z&sort_by=created_at&order=desc" \
  -H "Authorization: Bearer YOUR_API_KEY"
```
</CodeGroup>
