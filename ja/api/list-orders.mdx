## クエリパラメータ

<ParamField query="page" type="integer" default="1">
  ページ番号
</ParamField>

<ParamField query="per_page" type="integer" default="20">
  1ページあたりのアイテム数（最大: 100）
</ParamField>

<ParamField query="status" type="string">
  ステータスでフィルタ: `pending`、`processing`、`shipped`、`delivered`、`cancelled`
</ParamField>

<ParamField query="user_id" type="string">
  ユーザーIDで注文をフィルタ
</ParamField>

<ParamField query="from_date" type="string">
  この日付以降に作成された注文をフィルタ（ISO 8601形式）
</ParamField>

<ParamField query="to_date" type="string">
  この日付以前に作成された注文をフィルタ（ISO 8601形式）
</ParamField>

<ParamField query="min_total" type="number">
  最小注文合計金額
</ParamField>

<ParamField query="max_total" type="number">
  最大注文合計金額
</ParamField>

<ParamField query="payment_status" type="string">
  支払いステータスでフィルタ: `pending`、`paid`、`failed`、`refunded`
</ParamField>

<ParamField query="sort_by" type="string" default="created_at">
  ソートフィールド: `created_at`、`total_amount`、`status`
</ParamField>

<ParamField query="order" type="string" default="desc">
  ソート順: `asc`または`desc`
</ParamField>

<RequestExample>
```bash cURL
curl -X GET "https://api.example.com/api/v1/orders?status=processing&from_date=2024-01-01T00:00:00Z" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```python Python
import requests
from datetime import datetime, timedelta

url = "https://api.example.com/api/v1/orders"
headers = {"Authorization": "Bearer YOUR_API_KEY"}

# 過去30日間の注文を取得
from_date = (datetime.now() - timedelta(days=30)).isoformat()

params = {
    "status": "delivered",
    "from_date": from_date,
    "min_total": 10000,
    "sort_by": "created_at",
    "order": "desc",
    "per_page": 50
}

response = requests.get(url, headers=headers, params=params)
orders = response.json()

total_revenue = sum(order['total_amount'] for order in orders['data'])
print(f"過去30日間の売上: ¥{total_revenue:,}")
print(f"注文数: {orders['pagination']['total_count']}件")
```

```javascript JavaScript
// 本日の注文を取得
const today = new Date();

  `https://api.example.com/api/v1/orders?${params}`,
  {
    headers: {
      'Authorization': 'Bearer YOUR_API_KEY'
    }
  }
);

const data = await response.json();
console.log(`本日の注文: ${data.pagination.total_count}件`);
```

## 使用例

<CodeGroup>
```bash 処理中の注文
curl -X GET "https://api.example.com/api/v1/orders?status=processing" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```bash 特定ユーザーの注文
curl -X GET "https://api.example.com/api/v1/orders?user_id=user_456" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```bash 高額注文
curl -X GET "https://api.example.com/api/v1/orders?min_total=50000&sort_by=total_amount&order=desc" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```bash 期間指定
curl -X GET "https://api.example.com/api/v1/orders?from_date=2024-01-01T00:00:00Z&to_date=2024-01-31T23:59:59Z" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```bash 支払い失敗の注文
curl -X GET "https://api.example.com/api/v1/orders?payment_status=failed" \
  -H "Authorization: Bearer YOUR_API_KEY"
```
