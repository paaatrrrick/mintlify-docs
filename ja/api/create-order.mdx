## ボディパラメータ

<ParamField body="user_id" type="string" required>
  注文を行うユーザーのID
</ParamField>

<ParamField body="items" type="array" required>
  注文アイテムの配列
  <Expandable title="アイテムオブジェクト">
    <ParamField body="product_id" type="string" required>
      商品ID
    </ParamField>
    <ParamField body="variant_id" type="string">
      商品バリアントID（オプション）
    </ParamField>
    <ParamField body="quantity" type="integer" required>
      注文数量
    </ParamField>
    <ParamField body="price" type="number" required>
      単価
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="shipping_address" type="object" required>
  配送先住所の詳細
  <Expandable title="プロパティ">
    <ParamField body="postal_code" type="string" required>
      郵便番号
    </ParamField>
    <ParamField body="prefecture" type="string" required>
      都道府県
    </ParamField>
    <ParamField body="city" type="string" required>
      市区町村
    </ParamField>
    <ParamField body="address_line1" type="string" required>
      住所1
    </ParamField>
    <ParamField body="address_line2" type="string">
      住所2（建物名・部屋番号など）
    </ParamField>
    <ParamField body="recipient_name" type="string" required>
      受取人氏名
    </ParamField>
    <ParamField body="phone_number" type="string" required>
      電話番号
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="payment_method_id" type="string" required>
  支払い方法識別子
</ParamField>

<ParamField body="coupon_code" type="string">
  クーポンコード（オプション）
</ParamField>

<ParamField body="gift_wrapping" type="boolean" default="false">
  ギフトラッピングを希望
</ParamField>

<ParamField body="delivery_notes" type="string">
  配達メモ
</ParamField>

```python Python
import requests

url = "https://api.example.com/api/v1/orders"
headers = {
    "Authorization": "Bearer YOUR_API_KEY",
    "Content-Type": "application/json"
}
data = {
    "user_id": "user_456",
    "items": [
        {
            "product_id": "prod_789",
            "variant_id": "var_001",
            "quantity": 2,
            "price": 29999
        },
        {
            "product_id": "prod_790",
            "quantity": 1,
            "price": 19999
        }
    ],
    "shipping_address": {
        "postal_code": "150-0001",
        "prefecture": "東京都",
        "city": "渋谷区",
        "address_line1": "神宮前1-2-3",
        "address_line2": "ABCビル301",
        "recipient_name": "山田太郎",
        "phone_number": "090-1234-5678"
    },
    "payment_method_id": "pm_123",
    "coupon_code": "WELCOME2024",
    "gift_wrapping": True,
    "delivery_notes": "配達は午後2時以降でお願いします"
}

response = requests.post(url, json=data, headers=headers)
order = response.json()

print(f"注文番号: {order['id']}")
print(f"合計金額: ¥{order['total_amount']:,}")
print(f"配達予定日: {order['estimated_delivery']}")
```

<Note>
注文作成時に在庫の確認が自動的に行われます。在庫が不足している場合、注文は作成されません。
</Note>

<Tip>
送料無料の条件を満たしている場合、`shipping_fee`は0になります。
</Tip>
