## Why this documentation?

This documentation is built with [Mintlify](https://mintlify.com), offering:

- **Beautiful design** - Clean, modern UI that looks great on any device
- **Developer experience** - Code examples in multiple languages, copy buttons, and syntax highlighting
- **Fast search** - Find what you need instantly with built-in search
- **Easy maintenance** - Write in MDX and deploy automatically via GitHub
