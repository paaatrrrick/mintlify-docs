## Avant de Commencer

Avant de commencer le développement, assurez-vous d'avoir :

- Un compte valide et une clé API
- Node.js 18+ ou Python 3.9+
- Connaissance de base des APIs RESTful

## Exemple de Code

```javascript
// Exemple de démarrage rapide
const client = new APIClient({
  apiKey: 'votre_cle_api_ici'
});

const users = await client.users.list();
console.log(users);
```

```python
# Exemple en Python
from api_client import APIClient

client = APIClient(api_key='votre_cle_api_ici')
users = client.users.list()
print(users)
```
