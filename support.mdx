## Frequently Asked Questions

<AccordionGroup>
  <Accordion title="How do I get started?" icon="rocket">
    Check out our [Quickstart guide](/quickstart) to set up your documentation in minutes. You'll learn how to install the CLI, configure your docs, and deploy your first changes.
  </Accordion>
  <Accordion title="How do I customize my documentation?" icon="palette">
    You can customize colors, logos, fonts, and more in your `docs.json` file. See our configuration documentation for all available options.
  </Accordion>
  <Accordion title="How do I add API documentation?" icon="code">
    You can auto-generate API documentation from an OpenAPI specification. Simply add your OpenAPI file and reference it in your navigation configuration.
  </Accordion>
  <Accordion title="How do I deploy my changes?" icon="cloud-arrow-up">
    Install our GitHub app from your dashboard. Once connected, your changes will automatically deploy when you push to your repository.
  </Accordion>
</AccordionGroup>
