## 开始之前

在开始开发之前，请确保您有：

- 有效的账户和API密钥
- Node.js 18+或Python 3.9+
- RESTful API的基础知识

## 示例代码

```javascript
// 快速开始示例
const client = new APIClient({
  apiKey: '您的api密钥'
});

const users = await client.users.list();
console.log(users);
```

```python
# Python示例
from api_client import APIClient

client = APIClient(api_key='您的api密钥')
users = client.users.list()
print(users)
```
