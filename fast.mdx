Mintlify provides a modern documentation platform that's fast, beautiful, and easy to maintain.

## Key Features

  <Card title="Lightning Fast" icon="bolt">
    Built on Next.js with optimized static generation for instant page loads.
  </Card>
  <Card title="Beautiful by Default" icon="paintbrush">
    Professionally designed themes that make your docs look great out of the box.
  </Card>
  <Card title="Developer Focused" icon="code">
    Code blocks with syntax highlighting, multiple languages, and one-click copy.
  </Card>
  <Card title="Easy to Write" icon="pen">

  </Card>

## Built for Teams

- **Version control** - Your docs live in Git alongside your code
- **Automatic deployments** - Push to GitHub and your docs update instantly
- **Collaboration** - Multiple team members can contribute and review changes
- **Analytics** - Understand how users interact with your documentation
