## Antes de Comenzar

Antes de comenzar el desarrollo, asegúrate de tener:

- Una cuenta válida y clave API
- Node.js 18+ o Python 3.9+
- Conocimiento básico de APIs RESTful

## Código de Ejemplo

```javascript
// Ejemplo de inicio rápido
const client = new APIClient({
  apiKey: 'tu_clave_api_aqui'
});

const users = await client.users.list();
console.log(users);
```

```python
# Ejemplo en Python
from api_client import APIClient

client = APIClient(api_key='tu_clave_api_aqui')
users = client.users.list()
print(users)
```
