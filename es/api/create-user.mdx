## Parámetros del Cuerpo

<ParamField body="name" type="string" required>
  Nombre completo del usuario
</ParamField>

<ParamField body="email" type="string" required>
  Dirección de correo electrónico del usuario (debe ser única)
</ParamField>

<ParamField body="password" type="string" required>
  Contraseña del usuario (mínimo 8 caracteres)
</ParamField>

<ParamField body="bio" type="string">
  Biografía del usuario (opcional)
</ParamField>

<RequestExample>
```bash cURL
curl -X POST https://api.example.com/api/v1/users \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Juan Pérez",
    "email": "juan@example.com",
    "password": "contraseñasegura123",
    "bio": "Desarrollador de software en Madrid"
  }'
```

```python Python
import requests

url = "https://api.example.com/api/v1/users"
headers = {
    "Authorization": "Bearer YOUR_API_KEY",
    "Content-Type": "application/json"
}
data = {
    "name": "Juan Pérez",
    "email": "juan@example.com",
    "password": "contraseñasegura123",
    "bio": "Desarrollador de software en Madrid"
}

response = requests.post(url, json=data, headers=headers)
print(response.json())
```

```javascript JavaScript
const response = await fetch('https://api.example.com/api/v1/users', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    name: 'Juan Pérez',
    email: 'juan@example.com',
    password: 'contraseñasegura123',
    bio: 'Desarrollador de software en Madrid'
  })
});

const data = await response.json();
```
